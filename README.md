# 🧪 QA Manual Testing Practice - Login Module

## 📌 Descripción

Este repositorio contiene prácticas de testing manual enfocadas en el módulo de Login.

El objetivo de este proyecto es demostrar habilidades en:

- Diseño de casos de prueba
- Creación de checklists
- Reporte de bugs
- Organización de evidencia de testing

---

## 🎯 Alcance del proyecto

Módulo probado:

- Login de usuario

Tipo de pruebas realizadas:

- Pruebas funcionales
- Pruebas negativas
- Validaciones de campos
- Manejo de errores

---

## 🛠 Herramientas utilizadas

- Testing Manual
- Microsoft Excel
- Markdown
- Git & GitHub

---

## 📋 Casos de prueba incluidos

Se incluyen pruebas como:

- Login con credenciales válidas
- Login con contraseña incorrecta
- Login con campos vacíos
- Validación de mensajes de error

---

## 🐞 Bug Reports

Se documentan errores detectados durante la ejecución de pruebas, incluyendo:

- Pasos para reproducir
- Resultado esperado
- Resultado actual
- Severidad y prioridad
