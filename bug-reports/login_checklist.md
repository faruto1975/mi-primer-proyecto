# ✅ Login Checklist

## Verificaciones básicas

- [ ] El campo usuario acepta texto válido
- [ ] El campo contraseña acepta texto válido
- [ ] El botón Login funciona correctamente
- [ ] Se muestra mensaje si el usuario está vacío
- [ ] Se muestra mensaje si la contraseña está vacía
- [ ] Se muestra mensaje con credenciales incorrectas
- [ ] Permite login con credenciales válidas
- [ ] El sistema redirige correctamente después del login
- [ ] La sesión se mantiene activa
- [ ] El usuario puede cerrar sesión

## Seguridad

- [ ] La contraseña se oculta al escribir
- [ ] No permite SQL Injection básico
- [ ] No permite múltiples intentos sin control
