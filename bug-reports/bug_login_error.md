# Bug Report - Login Error

## ID:
BUG-001

## Título:
No se muestra mensaje al ingresar contraseña incorrecta

## Descripción:
Cuando el usuario ingresa una contraseña incorrecta, la página se recarga sin mostrar mensaje de error.

## Pasos para reproducir:
1. Ir a la página de login
2. Ingresar usuario válido
3. Ingresar contraseña incorrecta
4. Presionar el botón Login

## Resultado esperado:
Mostrar mensaje: "Contraseña incorrecta"

## Resultado actual:
La página se recarga sin mostrar ningún mensaje

## Severidad:
Media

## Prioridad:
Media

## Evidencia:
(Sin evidencia por el momento)
